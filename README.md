# Ott-Dashboard
Interactive Power BI dashboard analyzing Netflix, Amazon Prime Video, and Disney+ Hotstar. Features platform-specific dashboards, KPI cards, genre and ratings analysis, release trends, country-wise content distribution, and interactive slicers. Built using Power BI, Power Query, DAX, and Data Modeling.


📌 Features
Separate dashboards for Netflix, Amazon Prime Video, and Disney+ Hotstar
KPI cards for key metrics
Movies vs TV Shows analysis
Genre distribution
Ratings analysis
Release year trends
Country-wise content analysis
Interactive slicers and filters
Platform-specific dashboard themes


🛠️ Tools Used
Power BI Desktop
Power Query
DAX
Data Modeling
